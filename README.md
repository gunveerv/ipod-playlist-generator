# ipod-playlist-generator
a simple gui to manage ipod classic playlists, built with cpp

## engine
* select a folder
* check for music and playlist folders
* select a .m3u8 file and add/remove songs from music folder

## Precompiled
* a precompiled exe for silicon-based mac devices is added `mp3.out`

## engine future updates
* be able to select music and playlist folders

## commands
* `make && ./mp3.out ../ipod-files`
